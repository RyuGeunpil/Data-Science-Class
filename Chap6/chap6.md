<h2 style="text-align: center;">제6장 데이터의 시각화 </h2>

### 1.데이터의 시각화란?
- **데이터의 시각화:** 데이터의 정보를 보다 효과적으로 전달하기 위해서 그래프 등의 시각적 표현을 활용하는 것을 의미함

**데이터 형식과 시각화**  

| 데이터 형식                     | 예시           | 시각화 방법 예시                                      |
|----------------------------------|----------------|--------------------------------------------------------|
| **1변수<br>범주형**             | 성별/종교      | 막대그래프(Bar Chart), 원그래프(Pie Chart)            |
| **1변수<br>연속형**             | 연봉/키        | 히스토그램(Histogram), 상자그림(Box Plot)             |
| **2변수<br>범주/연속**          | 성별 & 성적    | 막대그래프(범주별), 상자그림(범주별)                  |
| **2변수<br>연속/연속**          | 키 & 몸무게    | 산점도(dot plot)                                      |
| **3변수<br>연속/연속/연속**     | 키 & 몸무게 & 성적 | 등치선도(contour plot)                           |



| 데이터 형식 | 예시           | 시각화 방법 예시                                      |
|-------------|----------------|--------------------------------------------------------|
| **1변수**   | 범주형: 성별/종교 | 막대그래프(Bar Chart), 원그래프(Pie Chart)            |
|             | 연속형: 연봉/키  | 히스토그램(Histogram), 상자그림(Box Plot)             |
| **2변수**   | 범주/연속: 성별&성적 | 막대그래프(범주별), 상자그림(범주별)                  |
|             | 연속/연속: 키&몸무게 | 산점도(dot plot)                                      |
| **3변수**   | 연속/연속/연속: 키&몸무게&성적 | 등치선도(contour plot)                         |




| 데이터 형식           | 예시          | 시각화 방법 예시              |
|:----------------------:|:-------------:|:-----------------------------:|
| 1개 변수   | 범주형    | 성별/종교       | 막대그래프, 원그래프 등      |
| 1개 변수   | 연속형    | 연봉/키       | 히스토그램, 상자그래프 등      |
